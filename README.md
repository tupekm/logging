# Test application for logging
This application is written ni Java. It is using Log4J 2 for logging

To use application clone or download zip file and import project in your IDE.

Maven application
Dependicies used:

```
		<dependency>
			<groupId>org.apache.logging.log4j</groupId>
			<artifactId>log4j-api</artifactId>
			<version>2.13.1</version>
		</dependency>
		<dependency>
			<groupId>org.apache.logging.log4j</groupId>
			<artifactId>log4j-core</artifactId>
			<version>2.13.1</version>
		</dependency>
```
